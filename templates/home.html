ALP - HUB

ALP is a knowledge-grounded conversational AI system.

The system enhances the accuracy of responses of GPT model related to a specific PDF document by using a retrieval augmentation technique.

You need an OpenAI API key to use ALP. You can get it here. Note that since ALP operates via localhost your API key is safe.
